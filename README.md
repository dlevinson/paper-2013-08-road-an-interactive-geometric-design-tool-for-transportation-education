# ROAD: An Interactive Geometric Design Tool for Transportation Education

This package contains the paper PDF and original ROAD Java app/source package for Liao and Levinson (2013), "ROAD: An Interactive Geometric Design Tool for Transportation Education," ASCE Journal of Professional Issues in Engineering Education and Practice 139(2):116-122. DOI: 10.1061/(ASCE)EI.1943-5541.0000142.

## Contents

- `paper/ROAD.pdf`: paper PDF.
- `code/original_java/applet/`: original ROAD applet launcher and JAR.
- `code/original_java/ROAD_SRC/`: original Java source, assets, manual, and demo files.
- `documentation/ARCHIVE_BOUNDARY.md`: audit decision and exclusion rationale.
- `metadata/SOURCE_FILE_DECISIONS.csv`: inclusion/exclusion decisions.
- `PACKAGE_MANIFEST.csv`: file-level package manifest.

## Survey Boundary

The paper reports class-evaluation survey results, but this package intentionally excludes raw response-level survey/evaluation files. The archive target here is the ROAD software and paper, not student survey data.
