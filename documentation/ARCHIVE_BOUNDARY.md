# ROAD Archive Boundary

Audit time: 2026-05-17 18:43:15 Australia/Sydney.

## Paper Evidence

The paper presents ROAD, a web-based geometric design and 3D visualization educational tool, and reports deployment/evaluation in CE3201. The paper describes the software tool as the primary research artifact; no public repository URL is provided in the text.

## Local Evidence

The NSF-STREET local source folders contain `RoadDesign.jar`, `ROAD.html`, and the `ROAD_SRC` Java source/assets/manual/demo files. The source folder on disk appears cloud-placeholdered, so this package uses the already materialized ROAD source staging copy in Code/Agents, which was itself copied from the NSF-STREET ROAD_SRC folder in an earlier pass. A later JavaScript modernization exists in the STREET-JS workspace and is documented as a related successor rather than duplicated here.

## Exclusions

Raw class-evaluation or response-level survey files are excluded because they are not required to archive the software and may carry privacy/ethics constraints. Manuscript drafts, review files, autonomous staging ledgers, and duplicate JS candidate folders are excluded from this package.

## Decision

Ready for public GitHub upload as a software package, subject to choosing the intended repository license.
